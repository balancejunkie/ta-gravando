# Testando Colchão com o Vendedor

**Episódio:** [[episodios/ep-337|Mande a Sua #337]]  
**Mecanismos:** #principio/escalacao #principio/constrangimento-social #principio/comedia-fisica  

## Setup
Ítalo vai a uma loja de colchões dizendo que quer comprar para ele e a esposa. Pede para o vendedor deitar junto para testar o peso de duas pessoas.

## Twist / Escalação
Começa com pedido razoável (testar peso), mas escala para pedir abraço, colocar perna por cima, e fazer posição de "papai" com o vendedor. O funcionário tenta manter profissionalismo enquanto recua cada vez mais.

## Melhores Reações
- "Aí pega não pô! Mas só para eu ver..."
- "Desculpa, mas ficou estranho pra mim"
- "Eu deito com você, eu faço tudo que eu tenho que fazer... mas aí tem um limite"

## Análise
A escalação gradual é a chave: cada pedido é marginalmente mais íntimo que o anterior, e o vendedor precisa decidir onde traçar a linha entre atendimento e constrangimento. O contexto comercial legítimo (testar colchão) justifica cada passo individualmente.
