# Mentira pra Esposa: Foto de Doido

**Episódio:** [[episodios/ep-348|Mande a Sua #348]]  
**Mecanismos:** #principio/escalacao #principio/constrangimento-social #principio/comedia-fisica  

## Setup
Ítalo recebe ligação de Alice (esposa ciumenta) e mente que está no culto. Pede para o estranho ao lado tirar foto 'rezando' ou fazer 'cara de doido' para mandar de prova.

## Twist / Escalação
As vítimas se envolvem nas mentiras cada vez mais elaboradas: rezar para foto, fazer cara de louco para fingir ser paciente psiquiátrico, dizer 'sou gay' para a esposa. A participação voluntária nas mentiras é o cerne cômico.

## Melhores Reações
- "Faz uma cara de doido, tipo crazy"
- "Fala assim: sou gay, sou gayizão"

## Análise
A cumplicidade involuntária das vítimas em mentiras conjugais cria humor por identificação — todos já pediram ajuda para uma mentirinha. A escalação das mentiras torna tudo absurdo.
